## 🔄 リアルタイム更新方法

### 実行中の更新
```bash
# 処理中にリアルタイムで追記
echo "$(date '+%H:%M:%S') - Processing image [N/M]: [image_name]" >> progress.md
```

### バッチ完了時の更新
```bash
# 最終結果の自動追記例
{
  echo "## 最終結果 ($(date '+%Y-%m-%d %H:%M:%S'))"
  echo "- 成功: ${SUCCESS_COUNT}枚"
  echo "- 失敗: ${ERROR_COUNT}枚"
  echo "- 成功率: ${SUCCESS_RATE}%"
} >> progress.md
```
